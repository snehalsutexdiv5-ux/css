Task 5: CSS Columns Example

This paragraph demonstrates auto columns. The browser decides how many columns fit based on the width of the container and the column width. It helps break long text into easy-to-read sections like a newspaper layout.

This paragraph demonstrates column-count. It is divided into 3 columns regardless of the container width. This is useful when you want a fixed number of columns.

Task 6: CSS Hyphens Example

Hyphens: none

This is an example paragraph with hyphens set to none. Long words will not be broken or hyphenated, which can cause overflow in narrow containers.

Hyphens: manual

This paragraph has hyphens set to manual. Only words with soft hyphen characters (­) will break. For example: hy­phen­ation.

Hyphens: auto

This paragraph uses hyphens set to auto. The browser automatically inserts hyphens at appropriate points when breaking lines in narrow containers, improving readability.
