Task 5: CSS Columns Example

This paragraph demonstrates auto columns. The browser decides how many columns fit based on the width of the container and the column width. It helps break long text into easy-to-read sections like a newspaper layout.

This paragraph demonstrates column-count. It is divided into 3 columns regardless of the container width. This is useful when you want a fixed number of columns.

This paragraph demonstrates column-width. Each column will be approximately 150px wide, and the browser decides how many columns fit inside the container.
